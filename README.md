# Canadian TV Simulator 2010s/2011

A nostalgic web-based simulator of Canadian television from the 2010s era. Watch Canadian channels with authentic programming schedules, commercial breaks, and retro UI.

## Features

- 🇨🇦 **Authentic Canadian Channels**: CTV, Global, TSN, Showcase, Muchmusic, YTV, and more
- 📺 **Dynamic Program Guide**: Real-time scheduling with authentic 2011-era programs
- 🎬 **Commercial Breaks**: Vintage Canadian commercials and bumpers
- 🎨 **Retro UI**: Authentic 2011 broadcast graphics and interface
- 🔊 **Sound Effects**: TV static, channel switching, and atmospheric audio
- 📅 **Weekly Schedule**: Full program listings and time-based scheduling

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

Then open your browser to `http://localhost:3000`
